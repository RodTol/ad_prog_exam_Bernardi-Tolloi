struct node$<$ T $>$}

Author\+: Rodolfo Tolloi, Giulia Bernardi.

Purpouse\+: create a custom class named node which implements a node for a binary search tree (B\+ST). Each node has two unique pointers, named right and left, and can have only one father\textquotesingle{}s node.

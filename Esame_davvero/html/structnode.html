Detailed Description

template<typename T>
struct node< T >

Author: Rodolfo Tolloi, Giulia Bernardi.

Purpouse: create a custom class named node which implements a node for a binary search tree (BST). Each node has two unique pointers, named right and left, and can have only one father's node.

Date: May, 2021.

Copy custom constructor. It creates a new node with the attribute of the starting node, but with the parent equal to a input pointer. The process is then iterated for all of its children, such that at the end, a deep copy of its subtree is performed
